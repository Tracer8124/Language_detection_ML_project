# 🌍 Language Detection using Machine Learning (20+ Languages)

## 📌 Overview
This project focuses on detecting the language of a given text using machine learning techniques. The model is trained to classify text into 20+ different languages based on textual patterns.

---

## 🚀 Features
- Detects 20+ languages
- End-to-end ML pipeline (preprocessing → training → prediction)
- High accuracy using NLP techniques
- Handles multilingual text inputs
- Clean and explainable workflow using Jupyter Notebook

---

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- NLP (CountVectorizer / TF-IDF)
- Jupyter Notebook

---

## 📊 Workflow

1. Data Collection
2. Text Preprocessing (cleaning, normalization)
3. Feature Extraction (TF-IDF / Bag of Words)
4. Model Training (Naive Bayes / Logistic Regression)
5. Evaluation (Accuracy, Confusion Matrix)
6. Prediction
